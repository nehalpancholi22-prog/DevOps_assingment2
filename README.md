# Python Assignment – Grade Checker, Dictionary & File Handling

This repository contains basic Python programs created as part of an academic assignment.
The programs demonstrate the use of conditional statements, dictionaries, and file handling.

---

## Programs Included

### 1. Grade Checker
- Takes marks as input from the user
- Displays grade using if-else conditions
- Grade system:
  - 90+ : A
  - 80–89 : B
  - 70–79 : C
  - 60–69 : D
  - Below 60 : F

### 2. Student Grades Using Dictionary
- Stores student names and grades using a dictionary
- Allows:
  - Adding new students
  - Updating existing grades
  - Viewing all student records
- Uses dictionary operations and conditional statements

### 3. Write to a File
- Creates a text file
- Writes content using file handling functions (`open`, `write`)

### 4. Read from a File
- Reads data from a text file
- Displays content using `file.read()`
